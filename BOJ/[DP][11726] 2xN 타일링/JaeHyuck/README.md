그럼 2*4 타일링은 2*2 타일링 2개와 (2*3타일링+ 2*1 타일링) 이런식으로 되는게 아닐까? 하고 문제를 풀기 시작했다.

처음엔

```python
import sys
input = sys.stdin.readline

num= int(input())

dp =[0]*100000
dp[1]=1
dp[2]=2
dp[3]=3
for i in range(4,num+1):
    dp[i]= dp[i-2]+3+dp[i-1]
print(dp[num])
```
